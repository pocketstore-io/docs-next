# Requirements

### NVM install
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
    source ~/.bashrc
    nvm install --lts

### Bun install
    npm i -g bun

### Pm2 install
    npm i -g pm2

### Snap install
    apt-get install snapd

### Go install
    snap install go --classic
