# Plugins

<hr style="border: 1px solid #1eadef;margin: 1rem 0">

**Clone**

    git clone pocketstore-io/plugin-hello-world.git

<p>
Getting Started clone the hello World Plugin, remove <b>.git</b>,
init git and push to your own Repo with a Commit.
<br><br>
Then you can start to make changes for your own Plugin,
after that you can publish your Repo to the Plugins Store, by PocketStore.
</p>

<hr style="border: 1px solid #1eadef;margin: 1rem 0">

**Folders & Files**

<p>
Plugins can contain multiple Files and Folder for Components, Pages, Layouts, Utils, Translations.
</p>

**Folders**

- **layouts** - Plugin Source Code
- **components** - Plugin Source Code
- **pages** - Plugin Source Code
- **translations**/{code}.json - Translation via code.json
- **utils** - Typescript utils

**Files**
- **plugin.json** - Plugin Source Code
- **schema.json** - Plugin Source Code
