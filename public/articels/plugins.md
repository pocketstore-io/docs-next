# Plugins

<hr style="border: 1px solid #1eadef;margin: 1rem 0">

**Clone**

    gh repo clone pocketstore-io/plugin-hello-world.git

<p>
Getting Started clone the hello World Plugin, remove <b>.git</b>,
init git and push to your own Repo with a Commit.
<br><br>
Then you can start to make changes for your own Plugin,
after that you can publish your Repo to the Plugins Store, by PocketStore.
</p>
